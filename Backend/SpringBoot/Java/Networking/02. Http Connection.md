Hasta Java 11, la forma estándar de realizar peticiones HTTP dentro de Java es utilizando el objeto *HttpURLConnection*.

**Nota: Solo tiene soporte para HTTP 1.1 y su implementación tiende a ser más *verbose* sobre *HttpClient*.**

Función *helper* para imprimir la información recibida:

```
public static void printResource(InputStream in) {
    try (BufferedReader reader = new BufferedReader(new InputStreamReader(in))) {
        String line;

        while((line = reader.readLine()) != null) {
            System.out.println(line);
        }
    } catch (IOException ex) {
        System.out.println(ex.getMessage());
    }
}
```

Petición GET:

```
public static void main(String[] args) {
    try {
        URI uri = URI.create("https://jsonplaceholder.typicode.com/posts/1");
        URL url = uri.toURL();

        HttpURLConnection connection = (HttpURLConnection) url.openConnection();
        connection.setRequestMethod("GET");
        connection.setRequestProperty("User-Agent", "Chrome");
        connection.setRequestProperty("Accept", "application/json, text/html");

        int code = connection.getResponseCode();
        System.out.println("code = " + code);

        if (code != HttpURLConnection.HTTP_OK) 
	        System.out.println("Error: " + connection.getResponseMessage());
        printResource(connection.getInputStream());
    } catch (IOException ex) {
        System.out.println(ex.getMessage());
    }
}
```

Petición POST:

```
public static void main(String[] args) {
    try {
        URI uri = URI.create("https://jsonplaceholder.typicode.com/posts");
        URL url = uri.toURL();
        
        HttpURLConnection con = (HttpURLConnection) url.openConnection();
        con.setRequestMethod("POST");
        con.setRequestProperty("Content-Type", "application/json; charset=UTF-8");
        con.setDoOutput(true);

        // Cuerpo JSON
        String jsonInputString = 
	        "{ \"title\": \"greeting\", \"body\": \"hello\", \"userId\": 1 }";

        // Escribir el cuerpo
        try (OutputStream os = con.getOutputStream()) {
            byte[] input = jsonInputString.getBytes("utf-8");
            os.write(input, 0, input.length);
        }

        // Leer la respuesta
        int code = con.getResponseCode();
        System.out.println("HTTP Response Code: " + code);

		printResource(connection.getInputStream());
    }
}
```
